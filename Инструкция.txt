Последовательность запуска, назначение программ, входные и выходные параметры.

1)
REC_make_txt_base.exe - создает приятно читаемую базу в текстовом формате. помогает в нахождении ошибок в базе

1.1)
выходные параметры(получаемые результаты и файлы после запуска программы)

1.1.1 файл in.txt

1.2)
входные параметры(способы запуска)

1.2.1 убедиться что в папке есть файл inn.txt. нажать 2 раза мышкой на значке программы.

1.2.2 перетащить файл с базой(с произвольным именем на значек программы)

2) REC_db_editor.exe - создание зашифрованной базы

2.1) выходные параметры
2.1.1 файл filo_db.rec

2.2) Входные параметры
2.2.1 убедиться что в папке есть файл in.txt. нажать 2 раза мышкой на значке программы.

3) REC_examenator.exe - задает вопросы сгенерированные из зашифрованной базы.

3.1) выходные параметры
3.1.1 файл с именем типа пользователь.exm

3.2) Входные параметры - нет


4) REC_reader.exe - программа проверки результатов теста

4.1) выходные параметры
4.1.1 обновленный файл с именем типа пользователь.exm
4.1.2 файл отчета с именем типа пользователь.txt

4.2) входные параметры - нет



уровни безопасности и предоставляемые файлы.

администратор (программист)
преподователь
пользователь

более верхний уровень имеет доступ ко всем файлам более нижнего уровня, но не наоборот.


REC_make_txt_base и  REC_db_editor - ТОЛЬКО администратор.

REC_reader.exe, незашиврованные текстовые базы - преподаватель

REC_examenator.exe, пользователь.exm, пользователь.txt - пользователь
